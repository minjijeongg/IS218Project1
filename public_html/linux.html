Linux Commands


Print Working Directory

The command below prints the current working directory:

$ pwd

List

The command below lists every contents in the directory:

$ list

Change Directory

The command below changes the working directory to [path]. You can press [Tab] key to auto complete the [path]

$ cd [path]

Make Directory

The command below creates a directory named [directory]:

$ mkdir [directory]

Remove

The commands below remove objects such as file, directory and more:

$ rm [file]
$ rm [directory]

Touch

The command below creates a file named [file]:

$ touch [file]

Visual

The command below opens a text editor for [file]. If there is no such file named [file], the command creates the file named [file] then opens an editor. More information about how to use vi is on vi tutorial

$ vi [file]

Nano
