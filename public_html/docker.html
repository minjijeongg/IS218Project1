Docker Commands


Docker hub = container registry = container store disk image

How to Build a New Version of your Image with your Updated Code

Use the command below:

$ docker build -t [username/bootstrapsite] .

How to Upload the Built Image from Local Machine to Docker Hub

Use the command below:

$ docker push [username/bootstrapsite]

How to List Containers

Use the command below:

$ docker ps

How to Run a Command in a New Container

Use the command below:

$ docker run -dp 80:80 [username/bootstrapsite]

How to Show what Contexts you are Using

Use the command below:

$ docker context show

Default = your local machine = your computer

How to Stop one or more Running Containers

Use the command below:

$ docker stop [container]

How to Remove one or more Containers

Use the command below:

$ docker rm [container]

How to Remove Unused Data

Use the command below:

$ docker system prune

How to Create and Start Container

Run containers in the background using the command below:

$ docker compose up
$ docker compose up -d

Type the following url in the browser:

localhost:8080

Insert the following contents in the 'Dockerfile' using vi commands:

FROM httpd:2.4                                   
COPY ./public_html/ /usr/local/apache2/htdocs/   

Insert the following contents in the 'docker-compose.yml' using vi commands:

version: "3.2"                                      
services:                                           
  apache:                                           
    build: .                                        
    ports:                                          
      - "8080:80"                                   
    volumes:                                        
      - ./public_html/:/usr/local/apache2/htdocs/
