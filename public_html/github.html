GitHub


What is GitHub?

GitHub is an open source version control system that allows users to keep track of changes to source code. The best part of using GitHub is that it allows you to work with multiple people.

Downloading and Installing Git Bash

  1. Download and install Git Bash
    • macOS
    • Windows
  2. Create a GitHub account
  3. Run using commands:
$ git config --global user.name [name]
$ git config --global user.email [name@example.com]

Setting up a Repository


Creating a New Repository
  1. Change directory to your project folder. Use Linux Command.
  2. Initialize a new repository using a command:
$ git init

How to Save Changes to the Repository: Git Add and Git Commit
  1. Change directories to [project path]
$ cd [path]
  1. Create a file to add to the repository
$ vi [file]
  1. Add to the repository staging area
$ git add [file]
  1. Check the status of added file
$ git status
  1. Create a new commit with a message
$ git commit -m [comment]

Cloning an Existing Repository

  1. Change directory to your project folder. Use Linux Command.
  2. Create a copy of remote repositories
$ git clone [repo_url]

How to Add Remote Repo to your Local Git Config

Use the command below:

$ git remote add [remote_name] [remote_repo_url]

How to Manage a Repository


How to Push Commits and Objects

Use the command below:

$ git push -u [remote_name] [local_branch_name]

How to Add a Branch

Use the command below:

$ git checkout -b [branch_name]

Pull Request

Pull requests are a mechanism for a developer to notify team members that they have completed a feature. Once their feature branch is ready, the developer files a pull request. It lets everybody involved know that they need to review the code and merge it into the main branch.
