Vi Commands


How to Start Vi

The command below opens the vi text editor:

$ vi [file]

How to Exit Vi

In order to quit vi and write out the file, follow the direction:

  1. Press [esc]
  2. Type either:
:x or :wq

In order to quit vi without saving, type either:

:q or :q!

How to Insert Text

The command below allows you to insert and add text:

:i

Press [Esc] to exit back to command mode.
