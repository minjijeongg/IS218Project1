Azure Commands


How to Create Student Azure Account

  1. Get the student edition of Azure
  2. Click the start free button
  3. Login to your student account or create microsoft student account using your school email address
  4. Go to microsoft azure website
  5. Locate the profile icon on the top right corner and click on the Azure portal link

Docker Azure Commands


How to Login to Azure Docker

Use the command below:

$ docker login azure

How to Create the Docker Context for Azure

Use the command below:

$ docker context create aci [context]

How to Switch to the Azure Context

Use the command below:

$ docker context use [context]

How to Release the Resources of Unused Containers, Networks, and Images while in the Azure Context

Use the command below:

$ docker system prune
